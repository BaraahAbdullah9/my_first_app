# مشروع تمارين التنقل وتمرير البيانات (Navigation Task)

هذا المشروع يحتوي على حل التمارين المطلوبة يوضح مفاهيم التنقل بين الشاشات في Flutter.

## التمارين المنجزة:

### 1. التمرين الأول: Basic Stack Navigation
- إنشاء شاشتين (Home & Details).
- استخدام Navigator.push للانتقال.
- استخدام Navigator.pop للعودة.

### 2. التمرين الثاني: Passing and Returning Data
- إنشاء شاشة قائمة منتجات وتمرير بيانات (الاسم، السعر، الوصف) إلى شاشة التفاصيل.
- إرجاع رسالة تأكيد عند العودة من شاشة التفاصيل.
- عرض النتيجة باستخدام SnackBar في الشاشة الرئيسية.


## ✍️ إعداد الطالبة:
- الاسم: براءة عبد الله
- المستوى: السنة الثالثة - هندسة برمجيات
